Filter Information

Filter codes are used internally to match filters across palettes, plans, and filter wheels. Common codes:

  • H - Hydrogen Alpha (Ha)
  • O - Oxygen III (OIII)
  • S - Sulfur II (SII)
  • L - Luminance
  • R, G, B - Red, Green, Blue
  • LP - Light Pollution

NINA mapping is configured separately in the filter wheel settings, allowing different physical filters to use the same filter code.
